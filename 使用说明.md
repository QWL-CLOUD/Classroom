# Classroom Sprint 2B 使用说明

## Library 资源库

- **+ Add resource**：新增资源。可填写类型、年级、技能、主题、单元、学习者、标签、链接和备注。
- **+ Collection**：建立文件夹式分类。删除 Collection 时，资源会移动到 Unfiled，不会被删除。
- **Favorites / Unfiled / Needs review**：左侧快速视图。
- **Select visible**：选择当前筛选结果，然后进行批量分类、收藏、标记已检查或删除。
- **Use in lesson**：把资源连接到已有课程。资源会记录使用次数和最近使用时间。
- **Find duplicates**：根据相同网址，或相同标题与类型，找出可能重复的资源。合并时保留信息最完整的一条，并合并标签、使用次数和课程链接。
- **Undo library action**：撤销最近一次 Library 修改，刷新页面后仍可撤销。
- **Export library**：单独导出 Library JSON 文件。完整备份仍可使用页面顶部的 Export backup。

## 本地隐私

所有个性化资料默认保存在当前浏览器 Local Storage，不会自动写入 GitHub。不要把导出的真实数据备份上传到公开仓库。
