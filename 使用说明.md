# Classroom Sprint 5 使用说明

## Teaching Memory 在哪里

Teaching Memory 不作为新的左侧主菜单。进入 **Learner Planning**，点击第三个标签 **Teaching Memory**。

## 添加记录

可以从四个位置添加：

1. Learner Planning 顶部的 **+ Teaching memory**；
2. Learners 中个人或小组卡片的 **Add memory**；
3. Individual Plan / Small-Group Plan 卡片的 **Memory**；
4. Weekly Planner 课程卡片的 **Memory**，或完成课程后点击提示中的 **Add reflection**。

记录可以包含观察、有效策略、挑战或误解、证据、下一步、复查日期、标签，并可关联课程和 Learner Plan。

## Follow-up

把状态设为 **Needs follow-up** 并填写 Review date。到期的记录会同时出现在 Teaching Memory 的 Review due 统计和 Automation 的 Review 队列中。

## 隐私与备份

Teaching Memory 只保存在当前浏览器和你连接的 Classroom Data 文件夹中，不会写入公开 GitHub。它会随完整 workspace 自动保存和备份。不要公开分享 Google Drive 中的 Classroom Data 文件夹。
