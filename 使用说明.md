# Classroom Sprint 6 使用说明

## Classroom Playbook 是什么

Classroom Playbook 是课堂常规与 SOP 资料库，不是每日课程表。

- **Daily Routines**：入班、晨间、过渡、收尾、离班等日常流程
- **Teaching Routines**：同伴交流、词汇教学、阅读讨论、写作反馈、形成性评价等教学流程
- **Operational Routines**：缺席补做、材料发放、设备管理、访客接待、紧急替课等操作流程

全班的日期与时间安排仍然保留在 **Weekly Planner**。

## 基本操作

1. 点击左侧 **Classroom Playbook**。
2. 在三个类别标签中选择一种 Routine。
3. 点击 **+ Add routine**。
4. 每一行填写一个 checklist step。
5. 可连接 Library 资源，也可把 Routine 连接到某一节已有课程。
6. 点击 **Run checklist** 可以临时执行清单；是否记录运行历史由你决定。

## 本地隐私

真实 Routine、资源连接、课程连接与运行记录只保存在浏览器和你连接的 Classroom Data 文件夹中，不会写入公开 GitHub 文件。

## 导入

可以在 Playbook 页面下载 CSV 模板，然后通过 Import Center 导入。导入时选择 **Classroom Playbook routines**，先检查字段映射与预览，再确认导入。
