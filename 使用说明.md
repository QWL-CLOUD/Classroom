# Classroom 使用说明（Sprint 2A.3）

## 更新 GitHub
将此文件夹内所有文件上传到仓库根目录。必须能在同一层看到：
- `index.html`
- `app-v2a3.js`
- `styles-v2a3.css`

## Undo Bump
在 Weekly planner 中点击课程卡片的 Bump，确认后会出现：
- 周计划工具栏中的 `Undo last bump`
- 页面右下角短暂显示的 `Undo`

撤销记录会保存在浏览器中，刷新页面后仍可使用。若 Bump 后手动修改了某节课的日期，撤销时会跳过该课程，避免覆盖手动修改。

## Import Center
左侧点击 `Import center`：
1. 选择 Excel、CSV、JSON 或 ICS。
2. 检查系统建议的分类。
3. 检查每一列的字段匹配。
4. 查看预览和警告。
5. 选择 Add、Merge 或 Replace 后导入。

真实资料只存入当前浏览器，不会自动进入 GitHub。
