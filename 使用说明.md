# Classroom Sprint 3 Corrected 使用说明

## 主要结构

- **Weekly Planner**：常规整班课程，周一至周日。
- **Learners**：只放需要额外跟踪的 Individuals 和 Small Groups，不放常规整班。
- **Learner Planning**：Individual Plans 和 Small-Group Plans。
- **Library**：教学资源索引。
- **Import Center**：本地导入个性化资料。
- **Storage & Backup**：连接 Classroom Data 文件夹。

## 搜索

搜索框固定在页面顶部。直接输入即可搜索课程、资源、学习者、小组、计划、Teaching Memory 等内容。Mac 按 Command + K，Windows 按 Ctrl + K，会直接把光标放进搜索框，不会跳转页面。

## 旧数据

原来的 One-on-one lesson 不会被删除：

1. 它仍保留在 Weekly Planner；
2. 系统会为它建立对应的 Individual Plan；
3. 不需要清除浏览器数据，也不需要重新连接 Google Drive 文件夹。

## 上传

把压缩包解压后的全部文件上传到 GitHub 仓库最外层。不要上传 ZIP 本身。
