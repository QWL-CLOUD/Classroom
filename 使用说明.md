# Classroom v8.1 使用说明

## 更新前
1. 在现有网站中先执行一次 **Backup now / Download backup**。
2. 不要清除 Chrome 的网站数据。
3. 解压新版 ZIP，把其中全部文件上传到 GitHub 仓库根目录。

## 导航
- 左侧展开时显示分类和图标。
- 点击折叠按钮后，只保留 **C** 和功能图标。
- 折叠后点击最上方的 **C**，导航会重新展开。

## Personal Agenda
Personal Agenda 只保存个人事项，例如预约、私人截止日期、差事和行政事项。固定课表、Calendar 服务、课堂 Session 和 Tasks 不再混入这个列表。

个人事项仍可在当天的 Today/Week 时间轴中显示，但 Personal Agenda 页面本身只列个人事项。

## Import Center
- 支持拖入多个文件。
- 单项勾选/取消不会回到列表顶部。
- 可一键全选当前可见、全选整份文件或全部取消。
- 普通 XLSX 可选择 Sheet、修改表头行并手动映射列。
- 未匹配列保留在来源数据中，不会静默删除。
- 导入 Schedule 或 Calendar 后，Today 和 Week 会立即刷新。

## PDF
文字型 PDF 会尝试提取并预览文字。扫描型 PDF 当前不会自动 OCR，只能作为本地参考文件保存，或在查看后手动建立 Calendar / Schedule 记录。
