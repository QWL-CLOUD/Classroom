# Classroom v9 使用说明

## 本地启动

在项目文件夹打开终端：

```bash
npm install
npm run dev
```

然后打开：

```text
http://localhost:5173/
```

## 四个相关视图

1. **Today Schedule**：查看当天真实时间轴。
2. **Edit schedule**：编辑当天事件、默认课表和日期例外。
3. **Open daily planning**：为当天每个教学时间块建立详细 Lesson Plan。
4. **Week**：Monday–Sunday 卡片式周计划。

## Today 数据关系

- **To-do** 与 **Tasks** 使用同一份数据。
- **Reminders** 来自 **Calendar**。
- Calendar 事件可以打开或添加到 Google Calendar。
- **Students to Notice** 来自当天 learner services 和 Teaching Memory。

## Learner Record

打开某个 Learner 后，可以查看：

- Profile
- Lesson Plans
- Planning Sessions
- Teaching Memory
- Services

Planning Session 是日期化的单独课程记录，可以同时出现在 Learner Record 和 Week 中。

## 数据安全

不要清除 Chrome 的网站数据。正式更新前先下载一次完整备份。Google Calendar 按钮目前是跳转/添加事件，不是自动双向同步。
