# Classroom 上传与使用说明

## 上传到你的 GitHub 私人仓库

1. 下载并解压 `classroom-app.zip`。
2. 打开你刚建立的 GitHub 私人仓库。
3. 点击 **Add file** → **Upload files**。
4. 把解压后的所有内容拖进去，包括：
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
   - `.gitignore`
   - `assets` 文件夹
5. 在提交说明中填写：`Add Classroom MVP`
6. 点击 **Commit changes**。

不需要安装 GitHub Desktop。

## 在电脑上直接打开

双击 `index.html` 即可在浏览器中运行。

## 当前版本包含

- 周一到周日的课程日历
- 普通课堂、一对一课程和备课三种类型
- 一对一 Session 独立页面
- 教学材料链接集中管理
- Bump 自动顺延：移动当前课程以及同一学习者后面的课程
- JSON 数据备份导出

## 数据保存位置

当前课程数据保存在浏览器本地。更换浏览器、清除浏览器数据或换电脑后，数据不会自动同步。请定期点击 **Export data** 保存备份。

下一阶段可以接入 Supabase 或 Firebase，实现账号登录、云端同步和多人协作。
