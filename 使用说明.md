# Classroom v8 使用说明

1. 将 ZIP 解压后的所有文件上传到 GitHub 仓库根目录。
2. 使用 Chrome 打开 GitHub Pages 网站，不要清除网站数据。
3. Import Center 支持拖拽多个文件；先选择或确认 Import destination，再预览和选择记录。
4. XLSX 视觉周课表会逐格识别时间段；普通活动汇总表可导入 Playbook、Library 等。
5. Schedule 编辑时可以选择多个 weekdays，并选择 This occurrence / future / entire default。
6. Calendar 中的 Social Worker、OT、Speech、Counseling 等服务会同步到 Today 的 Students to Notice。
7. Learners 页面打开个人档案，可查看该学习者自己的 Lesson Plans、Sessions、Memory 和 Services。
8. PDF 和图片可以预览并保存为本地参考记录。扫描件不会被系统静默猜测，需人工确认结构化内容。
