# Classroom — Sprint 5 Teaching Memory

Sprint 5 builds on Sprint 4 without changing the agreed left navigation or inline top search.

## Teaching Memory

- Lives inside Learner Planning as a third tab
- Records observations for selected individuals and small groups
- Captures what worked, challenges, evidence, next steps, review dates, tags, and follow-up status
- Links a memory to a learner plan and/or lesson
- Adds quick Memory actions from learner cards, learner plans, and Weekly Planner lesson cards
- Adds an Add reflection prompt after completing a learner-linked lesson
- Supports search, filters, sorting, pinning, resolve, edit, delete, export, and one-step undo
- Adds due teaching-memory follow-ups to Automation review
- Keeps all personalized data in the browser and connected Classroom Data folder

Whole-class teaching remains in Weekly Planner. Learners and Learner Planning remain focused on selected individuals and small groups.
