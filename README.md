# Classroom — Linked Workspace v9

A privacy-first local teaching workspace. Personalized data remains in the browser and the connected Classroom Data folder.

## Run locally with Vite

```bash
npm install
npm run dev
```

Open the address shown by Vite, normally:

```text
http://localhost:5173/
```

## Main workflow

- **Today**: Tasks, Calendar reminders, Students to Notice, Quick Capture, and the live daily schedule.
- **Edit schedule**: Events, default schedule, and date-specific overrides for the selected date.
- **Open daily planning**: A separate lesson-planning card for each teachable schedule block.
- **Week**: Monday–Sunday card view for lessons and important events.
- **Learners**: Open a long-term record for each individual or small group.

## Google Calendar

Calendar events have an **Open in Google Calendar** action. This hands the event to Google Calendar but does not perform automatic two-way synchronization.

## Updating an existing installation

Back up first. Replace the application files without clearing Chrome site data. The storage key remains `classroomDataV1`.
