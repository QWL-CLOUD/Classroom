# Classroom v9 — Linked Workspace

## Added

- Separate Today Schedule, Daily Schedule editor, Daily Planning, and Week card views.
- Today To-do reads and edits the same Tasks data.
- Today Reminders read Calendar events and open event details.
- Direct Google Calendar handoff for the calendar and individual events.
- Learner Record pages with Profile, Lesson Plans, Planning Sessions, Teaching Memory, and Services.
- Planning sessions are stored as dated sessions and remain available in Week/search.
- Card-based Monday–Sunday Week view with Pin, Bump, Duplicate, Delete, and Plan actions.
- Visible navigation expand rail when the sidebar is collapsed.
- Lesson plan fields for Standard, Learning Target, Learning Experience, and Evidence.

## Preserved

- Existing `classroomDataV1` browser data.
- Import Center, schedule exceptions, Bump/Undo, local folder backup, Library, Playbook, and personal Agenda.
- Personal Agenda remains separate from the recurring school schedule.

## Important boundary

“Open in Google Calendar” opens Google Calendar or prepares an event in Google Calendar. It is not bidirectional OAuth synchronization. Full two-way sync requires a Google API sign-in/backend phase.
